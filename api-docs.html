Introdução

API para geração e verificação de pagamentos via PIX, oferecendo uma solução completa para integração de pagamentos instantâneos.

Endpoints

Gerar Código PIX

Endpoint para gerar um novo código PIX para pagamento.

POST
https://api-production-0feb.up.railway.app/g5

Request Body

{
  "amount": 500,
  "description": "Pix",
  "customer": {
    "name": "string",
    "document": "string", // CPF válido
    "phone": "string",
    "email": "string"
  },
  "item": {
    "title": "string",
    "price": 500,
    "quantity": 1
  },
  "utm": "string"
}

Response

{
  "pixCode": "string",
  "transactionId": "string"
}

Verificar Status do Pagamento

Endpoint para verificar o status atual de um pagamento PIX.

POST
https://api-production-0feb.up.railway.app/verify5

Request Body

{
  "paymentId": "string"
}

Response

{
  "ok": boolean,
  "status": "pending" | "completed",
  "data": {
    "payment": {
      "customer": {
        "name": "string",
        "email": "string",
        "phone": "string",
        "document": "string"
      },
      "item": {
        "title": "string",
        "price": number,
        "quantity": number,
        "id": "string"
      },
      "status": "pending" | "completed",
      "paymentMethod": "pix",
      "amount": number,
      "fee": number,
      "result": number,
      "metadata": {
        "pixCode": "string",
        "pixQrCode": "string",
        "status": "WAITING_PAYMENT",
        "generatedAt": "string"
      }
    }
  }
}
